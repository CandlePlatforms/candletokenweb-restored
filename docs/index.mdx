# Introduction

Welcome! The pages that follow contain comprehensive documentation of the Candle
ecosystem. The documentation is split into two parts: Candle for
[consumers](/docs/video-developers/overview) and Candle for
[developers](/docs/products/overview). If you are new to Candle and interested
in how it works, you might want to check out the
[Protocol Overview](/docs/protocol/core-concepts/overview).

The documentation is an open-source community effort, so feel free to suggest
new topics, add new how-to guides, and provide examples wherever you think it
might be helpful. All documentation is editable via
[GitHub](https://github.com/CandlePlatforms/candle-org/tree/master/docs).
