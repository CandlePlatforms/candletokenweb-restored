# Overview

This getting started tutorial will help Products to:

- [Choose a role](/docs/products/getting-started/choosing-a-role) as a video
  miner
- Understand [hardware](/docs/products/reference/hardware) and
  [bandwidth](/docs/products/reference/bandwidth) requirements
- [Install](/docs/installation/install-candle) the node software
- [Connect](/docs/installation/connect-to-ethereum) to an Ethereum network
- [Activate](/docs/products/getting-started/activation) on the network
- [Test](/docs/products/getting-started/test) to ensure they can receive work on
  the network
- [Call reward](/docs/products/getting-started/calling-reward) to distribute
  CNDL rewards
